Question 2: Demonstrating the differrence between getElementById() and getElementsByTagName()

getElementById()

The difference is that GetElementByID() will retrieve a single element object based on the unique id specified, whereas getElementsByTagName() will retrieve an array of all element objects with the specified tag name. Here, on pressing the try button we'll particularly select an element by its particular id and modify it. In the output we'll see only the TESTING TEXT 1 turning into Green.




getElementsByTagName()

getElementsByTagName() retrieves an array of all element objects with the specified tag name. Then just like an array we can manipulate an individual element using it's index in the array. Here we will iterate over all the elements and turn their colour into Blue.
